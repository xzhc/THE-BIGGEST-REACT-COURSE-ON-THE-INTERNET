# 📚 学习笔记

> 问题驱动，自由记录

---

## 📂 目录结构

```
my-notes/
├── concepts/      # 概念和原理
├── projects/      # 项目分析
├── debugging/     # 问题记录
├── snippets/      # 代码片段
└── quick-reference.md  # 快速查询
```

---

## 💡 使用方式

**自由发挥** - 想记什么就记什么，想怎么记就怎么记

- 有疑问 → 直接问 mentor
- 有收获 → 随手记下来
- 有代码 → 保存到 snippets
- 需要回顾 → 查阅已有笔记

**没有模板限制，没有格式要求**

---

## 🔍 查找笔记

- 概念相关 → `concepts/`
- 项目相关 → `projects/`
- 问题相关 → `debugging/`
- 代码片段 → `snippets/`

或使用 VS Code 全局搜索（Ctrl+Shift+F）

---

**学习本身比笔记更重要** 🚀
